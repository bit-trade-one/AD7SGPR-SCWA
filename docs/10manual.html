簡易使用方法

AD7SGPRは文字列を送るだけで簡単に7セグを点灯できるデバイスです。
単なる表示だけでなくアニメーションも交えて表示することなども出来ます。
ターミナルソフトはもちろん、WEBブラウザからの制御にも対応しており、
Youtubeチャンネル登録者数などの表示も可能です。

本体各部名称

本体ボタン（A,B,C,Dの4つ）

それぞれを押すとシリアル通信でメッセージが返って来ます。

  • pushA
  • pushB
  • pushC
  • pushD

ボタンA:輝度設定

ディスプレイの輝度を0から7の範囲で変更します。
7の後は0になりループします。
デフォルトの輝度は5です。輝度0は無点灯ではありません。

ボタンB:7セグのON/OFF

7セグメントディスプレイを表示しないモードに入るor表示を消します。
表示はされないだけでAD7SGPRから応答は返ってきます。
もう一度押すと表示するモードに入ります

ボタンC:アニメーション設定

7セグ表示アニメーションを変更します。
アニメなし → 数字スロット風アニメ → セグメントぐるぐるアニメ → アニメなし

ボタンD:

全セグメントを点灯します。

送信コマンド

通信速度は115200bps 改行文字はCRLFが規定

コード 機能
英数字・記号(ASCII) 受信した英数字・記号を7セグディスプレイに表示します。(最大8桁まで、「.」は1つまで)
@CLR ディスプレイをクリアします。
@ANI[数字0-2] ディスプレイのアニメーションを設定します。
@ANI0 アニメなし
@ANI1 数字スロット風アニメ
@ANI2 セグメントぐるぐるアニメ
@7SG[数字0,1] ディスプレイの表示モードを設定します。
@7SG0 ディスプレイ非表示
@7SG1 ディスプレイを表示
7セグの表示はされないだけでAD7SGPRから応答は返ってきます
@BRI[数字0-7] ディスプレイの輝度を設定します。@BRI の後に 0 から 7 の数字をつけて輝度を設定します。例：@BRI5
@BRR 現在のディスプレイの輝度を返します。
@HEX[??*8桁] 16進数のデータで8桁のセグメントを個別に制御します。?HEX の後に 16桁の16進数をつけて表示します。
例：@HEXFFFFFFFFFFFFFFFFを送信すると、「8.8.8.8.8.8.8.8.」
@HEX0000000000000080を送信すると、「      .」
@HEX00000000000000FFを送信すると、「       8.」
@HEXFF00000000000000を送信すると、「8.       」

エラーコード内容

エラーコード エラー内容
E0 Over receive length
受信文字は20文字未満にしてください
E1 Multiple dots
ピリオドを1個以上存在させないでください
E2 Over 8 length
8文字を超えてしまっているので表示できない
E3 Need 16 hex digits
16桁の16進数が必要
E4 Brightness out of range
輝度が範囲外
